## 開発環境
* macOS Catalina 10.15.3 (ローカル開発環境)
* AWS Cloud9 Ubuntu 18.04 (クラウド開発環境）

## 使用技術

### バックエンド

* Ruby 2.5.5
* Rails 5.1.7
* Docker 19.03.5
* AWS Elastic Container Registry
* Docker-compose 1.16.1
* Git 2.17.1
* AWS EC2
* AWS RDS for MySQL 8.0.16
* AWS S3
* AWS VPC


### フロント
* SCSS
* Bootstarap 4.4.1
* font-awesome 4.7.0
* JQuery 2.1.3
* bulma 


## 機能一覧

* deviseを用いたユーザー登録&ログイン機能
* CarrierWaveを用いた画像のアップロード機能
* 投稿一覧・投稿詳細表示機能
* 投稿管理機能
* いいね機能
